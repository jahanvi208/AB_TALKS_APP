# Day 4 - Text Embeddings & Similarity

## What I Built
- Generated sentence embeddings using Sentence Transformers
- Compared sentence similarity using cosine similarity
- Took sentence inputs from the user
- Observed how similar meanings produce higher similarity scores

## Tech Used
- Python
- sentence-transformers
- scikit-learn

## Key Learning
AI does not understand words directly — it converts sentences into vectors (numbers). Similar meanings stay closer in vector space, which powers semantic search, recommendations, and modern NLP systems.
