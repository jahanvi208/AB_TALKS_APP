## How a Chatbot Works

A chatbot works by taking what the user types and turning it into something a computer can understand.  
The chatbot looks at the user's text input and breaks it down into smaller pieces before sending this information to a special computer program.  
This computer program is a language model that the chatbot has learned from.  
The language model looks for patterns in what the user said and comes up with a response that makes sense.  
Then the chatbot sends the response back to the user.
