# Day 7 — Sentiment Classifier 🚀

## 📌 Overview
This project is a simple Machine Learning model that classifies text as **Positive** or **Negative** using a Naive Bayes algorithm.



## Features
- Converts text → numerical features (Bag of Words)
- Trains a Naive Bayes classifier
- Predicts sentiment for user input
- Evaluates model accuracy



## 🛠️ Tech Stack
- Python  
- scikit-learn  
- CountVectorizer  
- Naive Bayes
