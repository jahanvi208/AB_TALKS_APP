# Day 6 — Machine Learning Intuition

## Project
Built a classifier using Google Teachable Machine to understand how machine learning models learn from data.

## Classes Used
- gang signs
- peace out sign

## What I Observed
1. The model became more accurate with more training data.
2. Consistent poses and lighting improved prediction quality.
3. Noisy or inconsistent data reduced model confidence and caused wrong predictions.

## Key Learning
Machine learning models learn patterns from examples, and model performance depends heavily on the quality and consistency of training data.
